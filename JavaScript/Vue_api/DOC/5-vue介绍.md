
## vue是一套构建用户界面的渐进式框架

vue.js + vue-router + vuex (状态管理) + axios

## 渐进式的理解

* 声明式的渲染(无需关系如何实现)
* 组件系统
* 客户端路由(vue-router)
* 大规模状态管理(vuex)
* 构建工具(vue-cli)

## Vue的两个核心点

* 响应的数据变化


## 安装vue

* cdn的方式: https://cdn.jsdelivr.net/npm/vue
* npm安装

```
//初始化
npm init     //不能有大写,特殊字符,中文,而且不要和安装的包的名字相同,
npm init -y

npm i vue --save   //默认下载最新版本

```
